# Introducción a la Guía del Desarrollador

Bienvenido a la Guía del Desarrollador de plugNmeet. Esta sección está diseñada para desarrolladores e integradores de sistemas que desean ir más allá de los complementos estándar e integrar profundamente plugNmeet en sus propias aplicaciones.

Ya sea que desee construir una solución de videoconferencia totalmente de marca blanca, crear una lógica de negocio compleja o contribuir al proyecto de código abierto, esta guía es su punto de partida.

## Lo que Puede Lograr

plugNmeet es más que una simple herramienta de videoconferencia; es una plataforma WebRTC flexible construida sobre una arquitectura moderna y orientada a la API. Con ella, puede:

*   **Construir una Experiencia Personalizada y de Marca Blanca:** Use nuestra API `getClientFiles` para incrustar y cambiar completamente la marca del cliente front-end de plugNmeet directamente dentro de su propia aplicación, sin quedar atrapado en un `iframe`.
*   **Automatizar y Gestionar Salas:** Cree, gestione y autentique usuarios para reuniones de forma programática utilizando nuestras completas API del lado del servidor.
*   **Ampliar la Funcionalidad con Webhooks:** Escuche eventos en tiempo real dentro de una reunión (como la unión/salida de un usuario, el inicio/detención de la grabación, etc.) para activar flujos de trabajo personalizados en su aplicación.
*   **Escalar para Audiencias Masivas:** Aprenda los conceptos arquitectónicos necesarios para construir una implementación distribuida y de alta disponibilidad capaz de soportar una base de usuarios masiva.

## Cómo Empezar

1.  **Explore la Introducción a la API:** Este es el mejor lugar para comenzar. Proporciona una referencia completa de todos los puntos finales del servidor disponibles y sus parámetros.
2.  **Revise la Guía de Personalización del Diseño:** Aprenda a usar un simple objeto de JavaScript para cambiar completamente la apariencia del cliente front-end.
3.  **Consulte nuestros SDKs:** Para una integración más rápida, proporcionamos SDKs oficiales para lenguajes populares como PHP y JavaScript.

¡Estamos emocionados de ver lo que construirá con plugNmeet!
